Questions

What is the difference between block tags and inline tags.

The best difference between the two is a separation, block tags will automatically have space between elements to separate them while the inline tags usually don't separate elements. For instance, an image tag is an inline element and this can overlap other elements if not set right.

What is the margin, border, and padding?

Margin is a space outside of the border and the border is what separates the element from the margin while the padding is a space inside the element that separates the border from the element.

What are units of measurement you can use in CSS?

You can use the absolute units: px, pt, cm, mm, in. Also the relative units can be used for measurement in css, these relative units are %, em, rem, vw, vh, vmin, vmx, ect.

When would you use a figure tag?

You use a figure tag when you need to separate figures which are usually images in the html.
