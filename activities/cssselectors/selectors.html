Engagement Activity

List out the different ways to select an item.

- An item can be selected in the html by using either of these: Tag name/Element selector, ID(#), Class(.), Attribute selector, descendant selector (div), child selector, etc.

What is the difference between an ID and a Class?

The difference between these two is a reusability, the ID tag is a unique label and the Class tag can be used multiple times.

What is the difference between a SPAN tag and a DIV tag?

- A span tag doesn't add breaks between the elements and it does not change anything outside of it while a "div" tag will affect the elements outside of it and add breaks above or below it.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

- The header, main, and footer tags are essential and the html will not be complete without them, also they categorize the entire html but a DIV tag is mainly used for managing layouts within a box and the html will be complete without it.

What are different ways to select a color?

- Selecting colors is possible through a selection of color names(color:;), hexadecimal code and rgb (both not covered yet).
