Web Development: Structural Tags

1. Structural Tags (HEADER, NAV, MAIN, FOOTER)

Header: "<header>" and </header>"

Navigation: "<nav>" and </nav>"

Main: "<main>" and "</main>"

Footer: "<footer>" and "</footer>"

2. What goes in the header?

- Contents that are usually preferred on top of the page, like titles and they don't change when the page is loaded.

3. What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

- Links to web pages are located here. A navigation can be placed in the header, it's common this way but doesn't have to be all the time. The Navigation might have a slight change when the "highlight" marker is displayed. And it can contain links for the internal web pages but for the outbounds can be possible but would not make sense to do that.

4. What goes in Main?

- Contents inside the main part of the web page that will always change when different web pages are loaded.

5. What goes in the Footer?

- Address, Contact, Copyright, and Policy Information are usually located here.

6. HTML Validation

It's a process of checking for tag errors, formatting errors for html, xhtml and etc.
