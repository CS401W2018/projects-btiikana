2. Explain the difference between an absolute link and a relative link. Why would you use relative links?

- While the absolute link can direct you straight to the main part or main page of the website, a relative link will help direct you to the sub parts or web pages within the main website. Relative links are useful for navigation to access pages within a website without always worrying about going to the main web page.
