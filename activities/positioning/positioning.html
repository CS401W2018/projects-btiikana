Questions

What is the difference between a block and an inline tag? Give examples of each one.

The difference between the two is one can separate other elements automatically while the other doesn't separate but might overlap other elements. For instance, the img tag will not create space automatically to separate elements but the link anchor will separate other elements.

What is normal flow in HTML?

A normal flow is a default layout of elements in the html. When elements aren't altered from their original formation or settings that is normal flow.

What does float do? Why do you need to clear the float?

A float will allow the inline and block elements wrap around each other or share room together.

What do absolute, fixed, and relative do?

Absolute places elements nearest to the elements' parents, fixed keeps the element visible when the window or display is changing and a relative will change the position of the elements' original spot but will still affect other elements.

How can you make nice navigation buttons for mobile devices?

You may use a fixed positioning so that the navigation will stay visible when users surf the web page without needing to go back up to the menus and also absolute positioning will be great for drop down menus.
