Why do we need a viewport meta tag?

  • We need a meta tag so that we can display our html content on the mobile devices.

    • When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

  • The one that is placed or written last will be displayed first.

    • How do you change the CSS for a web page displayed on a phone?

  • If you wanted to adjust for mobile devices viewport, you'd have to include "@media" on the css file and add any attribute to edit a web page display.

    • Notes

  • Adjust Media Screen on css file to 600px - 800px for ipad or mobile devices breaking points
