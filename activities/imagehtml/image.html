Questions

What kind of photos are good to encode using JPEG?

- The photos that are packed with colors and details with portraits and landscapes formats.

What kind of photos are good to encode using GIF?

- Photos with less details and fewer colors such as logos and aninmations.

What is unique about the PNG format?

- Images don't lose data or details when they're compressed and also this format allows for transparency option for the image.

How do you resize your photo to the size it should be displayed?

- Microsoft image viewers like a "photos" or any other software with the editing option can help resize the image.

Why should you resize photos using photo editing software instead of resizing it using CSS?

- It's more convenient to resize the image before using css as it the details will be retained when the image is loaded in the html. Css might affect the image and might cause some unwanted changes to the display quality when it's used to adjust size of the image.
