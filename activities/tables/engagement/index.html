Notes

  • tr tag - Separate items in the table to columns
  • th tag - Table Header
  • rowspan tag - when used it will interfere with a column below and a cell of the bottom column be will pushed to the right side so the bottom column td tag cell should be cleared.
  • colspan tag - will span a cell within a column entirely but there should be 1 colspan tag in a column as it will push the other td tags aside or forward if added.
  • theader, tbody tags - these can be used to purposefully style the contents within the table without needing to add classes individually.
  • tfoot tag - can be used for content like a "Total" at the bottom of the table
