Notes

  • Specific Tagging/Targetting - use a class and a comma before an element tag on the css file to target a specific content within the table. For example, (.class td, .class th..) and putting (.class td th) without a comma and without a class before it will target all of the th on the html

    • Questions

    Why don't we use the border property?

  • Because it's old school to use borders and also it looks cleaner when there's no borders.

    • What is the easiest way to color every other row a different color?

  • Using style property directly onto the html element can be done easily to color the rows but I personally think it's better to style the rows using css cos it's maintainable and manageable.

    • When should we use a table?

  • It's always only best to use tables when you need to display data that are natural to be displayed in a box like structure.
