Internet and Web Definitions


Internet

The Internet is an electronic system that grants connectivity access to all of the computer networks all over the globe.

Web

A Web is like a place that allows all of the information from the computer to be hosted and be shared by users online.

How are the Internet and Web different?

While the world wide web requires the internet access for hosting links, the internet can be used without links or web.

ASCII and Unicode

ASCII is an old set of instructions with limited characters in English characters and Unicode is more modern with more characters from different languages beside English. Which are used to tell a computer how to display letters, words and symbols.

What is UTF-8

A UTF-8 is a code that assists in informing computers to display and stack words and sentences correctly.

URL

The URL, Uniform Resource Locator, is an home address for a website on the internet.

Domain Name

A Domain Name is a unique name for a website address on the internet.

HTTP

HTTP stands for Hypertext Transfer Protocol, and it's a command that enables web browsers and websites with each other to communicate for purpose of showing you the pages you want to see.

HTTPS

HTTPS is a command just like HTTP but with added layer of security, that keeps information you send and receive private.

IP Address

An IP ADDRESS is an home address for a computer or other devices on the internet.

DNS

A DNS is like a phone book for the internet, that links website names to their proper IP addresses

W3C Recommendations

The W3C Recommendations are guidelines or commands used to display websites nicely and make them user friendly.

XML, XHTML, HTML, HTML5

The XML and XHTML are ways to organize information and structural data and HTML with HTML5 are special ways to build web pages with text and images.
